# 安装

## 环境支持

由于 Vue 3 不再支持 IE11，Element Plus 也不再支持 IE11 浏览器。

## 版本

IMM-UI 目前还在开发中

## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 IMM-UI, 然后您就可以使用打包工具，例如 Vite 和 webpack

```
# 选择一个你喜欢的包管理器

# NPM
$ npm install imm-ui --save

# Yarn
$ yarn add imm-ui

# pnpm
$ pnpm install imm-ui
```
